How to Use the To-Do List App

This smart and personalized to-do list web app is designed to help you manage your tasks efficiently. After logging in or signing up, you can access the dashboard, which includes the following features:

Add a Task

Create tasks with a title, description, date, time, deadline, and category (e.g., Reading, Writing, Poster, etc.). This data is saved per user and stored securely in your browser or Firebase.

Inbox

View all your pending (incomplete) tasks across all dates. You can complete, edit, or delete tasks here.

Today

Shows tasks due today based on the set date.

Upcoming

Displays tasks scheduled for future dates or deadlines.

Completed

Shows all tasks marked complete, grouped by date.

Filter

Search and filter tasks by title, date, category, and completion status.

Profile

Manage your full name, email, and password. Option to delete your account permanently.

Help & About

Read instructions on using the app or learn more about its mission and development.

Search Bar

Search your tasks live by typing in the dashboard’s search input.

Floating Add Button

A '+' button at the bottom-right opens the Add Task tab instantly.
